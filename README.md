# IT3213-P-Human-Computer-Interaction

## Overview

This repository contains practical work, assignments, and project materials developed for the **IT3213-P Human-Computer Interaction (HCI)** module.

The project focuses on designing user-friendly interfaces and improving user experience through modern HCI principles and prototyping techniques.

## Tools Used

* Axure RP
* Figma
* Microsoft Office
* Draw.io

## Topics Covered

* Introduction to Human-Computer Interaction
* User-Centered Design
* Usability Principles
* User Experience (UX) Design
* Interface Design
* Wireframing and Prototyping
* User Evaluation Techniques
* Accessibility and Inclusive Design

## Prototype Development

The system prototype and user interface designs were developed using **Axure RP**.

Features included in the prototype:

* User Registration and Login Screens
* Navigation Flow Design
* Interactive UI Components
* Form Validation Design
* Dashboard Interfaces
* User Feedback Screens


## Software Requirement

* Axure RP 10 or later
* Windows 10/11 or macOS

## Learning Outcomes

After completing this module, students will be able to:

* Design user-friendly interfaces.
* Apply usability principles in system design.
* Develop interactive prototypes using Axure RP.
* Evaluate user experience and interface effectiveness.
